OpenLayer - Open Source Photoshop ComfyUI Plugin

OpenLayer is an open-source Photoshop UXP plugin for local ComfyUI text-to-image, image-to-image, and sketch-to-image workflows. Generate, reinterpret an active layer, guide a result with LineArt ControlNet, then preview and import the result as an editable Photoshop layer.

Artist-first foundation

A focused first step for local image generation inside Photoshop.

OpenLayer starts with dependable local workflows: text-to-image generation, image-to-image, a first Sketch to Image LINECN path, canvas capture, panel previews, and import into the active document. The goal is a clean base that can grow into deeper layer, mask, and selection workflows without rushing fake features.

MVP features

What works now

Home dashboard

Photoshop-dark tool cards expose Text to Image, Image to Image, History, Settings, and planned future tools without fake implementation.

ComfyUI connection

Set a local server URL, find common active local ports, check availability, and load checkpoints from ComfyUI.

Text-to-image workflow

Send prompt, negative prompt, size, steps, CFG, seed, and checkpoint into a starter workflow.

Image-to-image foundation

Capture the active Photoshop layer or visible canvas, preview the source, upload it to ComfyUI, and generate through a starter img2img workflow.

Sketch to Image LINECN

Capture a layer or canvas, run a starter SD 1.x LineArt ControlNet workflow, preview the result, and import it back to Photoshop.

Experimental checkpoint mode

Keep SD3 and Flux-style checkpoints visible with compatibility warnings while keeping img2img-basic focused on SD 1.x and SDXL.

Panel preview

Retrieve generated output through ComfyUI history and view endpoints, then show it in the panel.

Photoshop layer import

Place the generated image into the active document manually, or enable optional auto-import after generation.

Settings and history

Remember local defaults and review recent generated previews during the current panel session.

GPU-aware recommendations

Detect the GPU and VRAM reported by ComfyUI, then show beginner-friendly model and workflow guidance in Settings.

Workflow compatibility foundation

Separate checkpoint-based presets from future diffusion-model-stack presets such as Z_image_Turbo and Flux without enabling unvalidated workflows.

UXP-safe interface

Built with vanilla TypeScript and responsive spacing rules that avoid browser APIs and layout features that are fragile in Photoshop UXP.

Future-ready structure

ComfyUI, Photoshop, UI, workflow, and utility modules are separated for careful next steps.

The MVP loop in Photoshop

Workflow

How it works

  1. Open Photoshop and load the OpenLayer UXP panel.
  2. Run ComfyUI locally on the OpenLayer server port.
  3. Use Settings to check ComfyUI and optionally detect GPU/VRAM recommendations.
  4. Choose Text to Image, Image to Image, or Sketch to Image.
  5. Choose a compatible checkpoint and enter a prompt.
  6. Generate through the ComfyUI prompt API.
  7. Preview the output image in the panel.
  8. Import the result as a new named Photoshop layer.

Setup

Run the MVP locally

1

Install and build

npm install
npm run build
2

Load in UXP Developer Tool

Select the generated dist/manifest.json, click Load, then open the OpenLayer panel in Photoshop.

3

Start ComfyUI

python main.py --listen 127.0.0.1 --port 8190
4

Create the first layer

Find or check ComfyUI, optionally detect GPU recommendations, choose a checkpoint, generate, preview, and import the result into the active document.

OpenLayer defaults to http://127.0.0.1:8190 so it can run beside other local tools that may already use port 8188.

Known limitations

Honest alpha boundaries

  • txt2img-basic, img2img-basic, and sketch2img-linecn-basic are starter workflows, not final production presets.
  • Image to Image and Sketch to Image source capture now encodes raw Photoshop pixels as PNG/lossless images.
  • Sketch to Image currently targets SD 1.x LINECN with epicrealism_naturalSinRC1VAE.safetensors and control_v11p_sd15_lineart_fp16.safetensors.
  • Custom ComfyUI workflows may need node ID updates in the preset registry.
  • GPU-aware recommendations are advisory only. OpenLayer does not auto-switch models or workflows yet.
  • Generated images are placed as Photoshop layers; advanced alignment is future work.
  • Inpaint is experimental: source and mask export are available, but output quality and Photoshop alignment are still being tested.
  • The browser dev server is only for UI preview. Photoshop APIs run inside UXP.
  • Responsive panel spacing has been improved for testing, but final visual polish is still in progress.

Tester checklist

Pre-release smoke test

  1. Start ComfyUI on http://127.0.0.1:8190.
  2. Load dist/manifest.json in Adobe UXP Developer Tool.
  3. Open Photoshop, launch OpenLayer, and click Check ComfyUI.
  4. Generate one text-to-image result and import it as a new layer.
  5. Capture an active layer or canvas, run Image to Image, and import the result.
  6. Capture an active layer or canvas, run Sketch to Image with sketch2img-linecn-basic, and import the result.
  7. Resize the panel narrow and wide; confirm cards, buttons, previews, header, and footer remain reachable.
